Sifteo API Demos

These sample programs are provided to illustrate various parts of the Sifteo API. Sifteo's goal is to provide a framework that is robust enough for the most complex projects, but simple enough for rapid prototyping and sketching.

01. Hello Sifteo

This demo illustrates the basic structure of a Sifteo app, and demonstrates some of the basic drawing functions.

The following interfaces and concepts are demonstrated in "Hello Sifteo:"

  • BaseApp
  • BaseApp.Setup and BaseApp.Tick
  • Cube and CubeSet
  • Color
  • Cube.FillScreen and Cube.FillRect
  • Cube.Paint

02. Slide Show

This demo illustrates the event model for Sifteo apps, the gestures you can listen for, and how to draw an image to a cube's screen.

The following interfaces and concepts are demonstrated in "Slide Show:"

  • Cube.Image
  • Cube gesture events:
    • NeighborAddEvent
    • NeighborRemoveEvent
    • ButtonEvent
    • TiltEvent
    • ShakeStartedEvent
    • ShakeStoppedEvent
    • FlipEvent
  • Wrapper pattern

This app can also serve as a utility for quickly testing graphics on a cube.

03. Sorter

This demo illustrates the basics of making an app robust with pause/unpause and new/lost cube handlers; simple audio; and utility functions for dealing with networks of neighbored cubes.

The following interfaces and concepts are demonstrated in "Sorter:"

  • BaseApp and CubeSet events:
    • PauseEvent
    • UnpauseEvent
    • NewCubeEvent
    • LostCubeEvent
  • Sounds.CreateSound, Sound.Play, Sound.Stop
  • CubeHelper.FindRow, CubeHelper.FindColumn, and CubeHelper.FindConnected
  • More on Image rotation

04. Flocker

This demo illustrates objects in motion on the cubes. It also demonstrates ways to check sensor state (neighbors, tilt, button) without using event handlers.

The following interfaces and concepts are demonstrated in "Flocker:"

  • BaseApp.DeltaTime and BaseApp.FrameRate
  • Checking sensors by state rather than event:
    • Cube.Neighbors
    • Cube.ButtonIsPressed
    • Cube.Tilt
    • Cube.IsUpright
    • Cube.IsShaking

Source code for Flocker is split into multiple files for easier reading:

  • FlockerApp.cs contains the BaseApp.
  • FlockerWrapper.cs contains the cube wrapper.
  • FlockerShape.cs contains the shapes that fly around.
